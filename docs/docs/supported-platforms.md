# Supported platforms

| Platform | Version                |
| -------: | :--------------------- |
|      iOS | >= 10.0                |
|    macOS | >= 10.14               |
|  Android | >= 22                  |
|    Linux | == Ubuntu 20.04 x86_64 |
|  Windows | >= 10 x86_64           |

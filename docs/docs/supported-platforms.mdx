# Supported Platforms

:::note

Couchbase Lite for Dart requires Dart SDK `^3.10.0`.

:::

| Platform | Version                |
| -------: | :--------------------- |
|      iOS | >= 10.0                |
|    macOS | >= 10.14               |
|  Android | >= 22                  |
|    Linux | >= Ubuntu 20.04 x86_64 |
|  Windows | >= 10 x86_64           |

## Default database directory

When opening a database without specifying a directory, Couchbase Lite uses the
platform's standard app data directory for mobile and deployed desktop
applications. When running during development (e.g. via `dart run`), the current
working directory is used. `api|Database.defaultDirectory` allows you to specify
a different default directory.
